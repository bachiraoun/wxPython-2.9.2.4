enum wxDataViewCellMode

The mode of a data-view cell; see wxDataViewRenderer for more info.

Enumerator:
wxDATAVIEW_CELL_INERT 
wxDATAVIEW_CELL_ACTIVATABLE 

Indicates that the user can double click the cell and something will happen (e.g.

a window for editing a date will pop up).

wxDATAVIEW_CELL_EDITABLE 

Indicates that the user can edit the data in-place, i.e.

an control will show up after a slow click on the cell. This behaviour is best known from changing the filename in most file managers etc.
